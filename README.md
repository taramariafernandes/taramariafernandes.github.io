# taramariafernandes.github.io

Personal landing page for Tara Maria Fernandes — PhD, Teachers College Columbia University.

## Setup

### 1. Create the repository
- On GitHub, create a **new repository** named exactly: `taramariafernandes.github.io`
- This special name tells GitHub to publish it as your personal site

### 2. Upload the file
- Upload `index.html` to that repository (drag and drop on the GitHub website, or use git)

### 3. Enable GitHub Pages
- Go to **Settings → Pages**
- Under "Source", select **Deploy from a branch**
- Branch: `main` / Folder: `/ (root)` → Save

Your site will be live at: **https://taramariafernandes.github.io** within a minute or two.

## Personalise

Before publishing, update these placeholders in `index.html`:

| What | Where to find it |
|------|-----------------|
| Your photo | Replace the grey placeholder `<div>` with `<img src="your-photo.jpg" alt="Tara Maria Fernandes" />` and upload the photo to the repo |
| Email | Search for `your.email@columbia.edu` and replace |
| LinkedIn URL | Search for `linkedin.com` placeholder link |
| Undergraduate institution | Search for `Add your undergraduate institution` |
| BA year | Add the year next to your BA entry |

## Add more sections later

Want to add a **Presentations** or **Teaching** section? Copy one of the existing `<section>` blocks, give it a new `id`, and add a matching link in the `<ul class="sidebar-nav">`.
